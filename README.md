Senior UX/UI Developer – Take‑Home Exercise
Project: Agentic Developer Portal – Git Repository Workflow + Agent Execution UI
Overview
Your task is to design and build a small slice of an AI‑native Developer Portal. The focus is on how developers interact with repositories, trigger AI‑powered automation, and monitor real‑time agent activity. You can use mock data throughout the exercise.

Part 1 — UX Design
Create wireframes that illustrate a simple end‑to‑end workflow where a developer can:
Choose a repository
View basic insights about that repo
Trigger an AI Agent task (e.g., create a PR, refactor code, upgrades, view dependency, run tests, scan etc)
Watch the agent’s progress update in real time
Deliverables:
Low‑ or mid‑fidelity wireframes
A 1–2 page write‑up explaining the reasoning behind your UX decisions


Part 2 — Frontend Implementation (React + TypeScript)
Build a small UI that demonstrates the following:
1. Repository List
Mocked list of repositories
Search and filtering
Ability to select a repo
2. Repository Overview
Basic metadata and insights
A “Run Agent Task” action
3. Agent Execution Panel
Simulated streaming logs
Status updates (pending → running → success/failure)
Retry option + a details modal or drawer
Technical Requirements
React + TypeScript
Mock data + mock streaming behavior
Clean, modular component structure
